🚀 Excited to share my latest project: **SAC Code Prediction System**

Selecting the correct SAC (Service Accounting Code) manually can be confusing and time-consuming due to thousands of similar service categories.

💡 To solve this, I built an **AI-powered system** that automatically predicts the most relevant SAC code from a given service description using **Natural Language Processing (NLP)**.

🔍 **What this project does:**
✔ Converts service descriptions into embeddings using Sentence Transformers
✔ Uses cosine similarity to find the best match
✔ Provides real-time predictions via FastAPI
✔ Offers an interactive UI using Streamlit

🛠 **Tech Stack:**
Python | FastAPI | Streamlit | Sentence Transformers | Scikit-learn | Pandas

📊 **Key Impact:**
• Reduces manual effort in classification
• Improves accuracy using semantic understanding
• Demonstrates real-world NLP application
